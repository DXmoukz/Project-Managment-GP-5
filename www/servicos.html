Faqs

Q1: O que é um portfólio?

R1: Um portfólio consiste num relatório e nos seus ficheiros adicionais de um projeto de final de curso ou estágio, podendo conter vídeos, fotos e documentos adicionais (word, pdf, pptx, etc.)

Q2: Como posso fazer o registo no site?

R2: Na página inicial do site existirá uma aba no canto superior direito cujo nome é “Iniciar Sessão”, ao clicar será redirecionado para a pagina de inicio de sessão, caso já tenha uma conta bastará iniciar a sessão com os seus dados pessoais, caso contrário, existe um link na pagina de login de seu nome “Crie uma conta” que abre a pagina de registo do site, bastará preencher os campos necessários e criar a sua conta.

Q3: Como posso aceder a um portfólio?

R3: O site tem na sua página inicial uma aba “Portfólios” que abre uma página com todos os portfólios presentes no site, do lado esquerdo existirá uma zona de pesquisa com as áreas científicas dos portfólios, que ajudará a filtrar os portfólios a mostrar.

Q4: Como posso publicar um portfólio?

R4: Este processo pode ser feito de duas maneiras, diretamente a partir da pagina principal onde existirá uma aba dedicada para esta função. A segunda forma será depois de ser iniciada sessão, haverá um menu chamado “Criar portfólio”, esse menu terá uma ligação para a pagina com todos os requisitos para a criação de um portfólio.

Q5: Como posso contactar o suporte?

R5: A partir da pagina inicial haverá uma aba no menu chamada de “Contactos”, esta aba irá remeter para a pagina que contem todos os contactos do site.
